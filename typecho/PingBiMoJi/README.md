# PingBiMoJi - 笔墨迹主动推送插件

发布文章时自动通知笔墨迹抓取，实现文章实时同步。

## 安装方法

1. 将 `PingBiMoJi` 文件夹上传到 Typecho 的 `usr/plugins/` 目录
2. 登录 Typecho 后台，进入「控制台」→「插件」
3. 找到「笔墨迹主动推送插件」，点击「启用」
4. 点击「设置」进行配置

## 配置说明

| 配置项 | 说明 | 示例 |
|-------|------|------|
| 笔墨迹 Ping 接口地址 | 默认无需修改 | `https://blogscn.fun/blogs/api/ping` |
| 你的 RSS 地址 | **必填**，你在笔墨迹登记的 RSS 地址 | `https://yourblog.com/feed/` |
| 验证 Token | 可选，如果笔墨迹分配了 Token | 留空即可 |
| 是否启用推送 | 启用/禁用推送功能 | 启用 |
| 推送时机 | 仅新发布 或 发布和更新时 | 仅新发布时 |
| 调试模式 | 开启后记录详细日志 | 关闭 |

## 使用方法

配置完成后，每次发布新文章时，插件会自动：

1. 向笔墨迹发送 Ping 通知
2. 笔墨迹收到通知后立即抓取你的 RSS
3. 新文章会在几秒内出现在笔墨迹

## 频率限制

- 每个博客每小时最多 Ping 5 次
- 超过限制会返回 429 错误

## 常见问题

### Q: 推送失败怎么办？

1. 检查 RSS 地址是否正确
2. 确认你的博客已在笔墨迹登记
3. 开启调试模式查看详细错误

### Q: 为什么文章没有同步？

1. 检查文章是否已发布（草稿不会触发）
2. 检查推送时机设置
3. 查看插件设置页面的「最近推送日志」

### Q: 支持哪些 Typecho 版本？

支持 Typecho 1.2.0 及以上版本。
